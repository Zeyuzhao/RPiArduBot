RPiArduBot Home

Welcome to the RPiArduBot site.

RPiArduBot is an open source program that is used for controlling robots with a Raspberry Pi and an Arduino.

It is written in the languages of Python, HTML/CSS, Javascript, and C++.

Learn more by clicking one of the links in the navigation bar above.



Source Code

 The source code is available here, but can also be downloaded from the Github repository.


Features


Control Methods.

Arrow Pad

 Use a basic 4-direction virtual arrow pad to control the robot


Joystick

 Use a virtual joystick with unlimited directions and a built-in speed control


Integrated camera feed.

The camera feed from the robot, if applicable, can be viewed from within the program, and has a high compatibility.



Fast setup

Very little installing is required, and the program can be run directly from the downloaded folder.



Streamlined UI

The user interface is very streamlined, modern, and easy to use.
